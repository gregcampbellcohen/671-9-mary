Gray's Arch and 717 named arches

University of Kentucky Geography

Arches extracted from the Domestic and Antarctic Names (State and Topical Gazetteer) published by the United States Board on Geographic Namkes. (geonames.usgs.gov) Data accessed August, 2018. Contours were extracted from a USGS NED 1/3 arc-second raster digital elevation model. This data can be accessed at the USGS 3D Elevation Program (usgs.gov)

Tools used in this project

  • PostGIS was used to store/query the national dataset.
  • Data is shown on Mapbox web map with geolocation services enabled. If the map got popular it would require a fee to use.
  • GDAL and QGIS 3.2 to convert raster elevation units and create contours.
